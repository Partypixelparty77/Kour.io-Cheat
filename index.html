Download & Install

Installation Guide

  1. Install a Userscript Manager: You need an extension to manage userscripts. We recommend Tampermonkey, available for Chrome, Firefox, Safari, and Edge.
  2. Click the Download Button: Press the "Download & Install" button located above on this page.
  3. Confirm Installation: Your userscript manager will open in a new tab. Review the script details and click the "Install" button.
  4. Launch the Game: Navigate to kour.io. The script is now active. Press the 'o' key to open and close the Orbit menu.
